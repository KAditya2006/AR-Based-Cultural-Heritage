This directory contains all image assets for the AR Cultural Heritage Platform.

- `favicon.ico` - Browser favicon (16x16, 32x32, 48x48px)
- `hero-bg.jpg` - Hero section background (1920x1080px)

## Image Guidelines

### Format Recommendations
- **Photos**: JPEG format for photographs
- **Graphics**: PNG format for logos and graphics with transparency
- **Icons**: SVG format for scalable icons

### Size Guidelines
- **Hero Images**: 1920x1080px (16:9 aspect ratio)
- **Preview Cards**: 400x300px (4:3 aspect ratio)
- **Avatars**: 100x100px (1:1 aspect ratio)
- **Icons**: 24x24px, 32x32px, 48x48px

### Optimization
- Compress images for web delivery
- Use WebP format where supported
- Provide multiple sizes for responsive images
- Include alt text for accessibility

## Usage Notes

All images should be optimized for web delivery and include appropriate alt text for accessibility. Consider using a CDN for better performance in production.

For development, you can use placeholder images from services like:
- Unsplash (https://unsplash.com)
- Pexels (https://pexels.com)
- Lorem Picsum (https://picsum.photos)

Remember to replace placeholder images with actual heritage site photographs and proper licensing before production deployment.
